**Word Counter** is a C program that takes a sentence as input from the user and counts the number of characters, words, and lines present in the text. It uses the fgets() function to read the input and processes each character using a loop. Characters are counted directly, words are identified based on spaces, tabs, and newline separators, and lines are counted whenever a newline character is encountered. The program demonstrates basic concepts of C programming such as strings, loops, conditional statements, and input handling.
